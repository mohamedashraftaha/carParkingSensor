# carParkingSensor
Simple Car Parking Sensor using STM32 NUCLEO-L432KC and HC-SR04 ultrasonic sensor.

# Hardware Components
* STM32 NUCLEO-L432KC
* Passive Buzzer
* HC-SR04 Ultrasonic Sensor
* Breadboard
* Jumper Wires
